\section{Artificial Intelligence}

\subsection{Gemini Models from Google AI}

Gemini is a family of large language models developed by Google DeepMind, designed to support multimodal understanding including text, code, and structured data. These models are optimized for reasoning tasks, natural language understanding, and software engineering-related applications.

In the CodeRank system, Gemini is used as the core AI engine powering the chatbot and coding interview assistant. The system primarily leverages the text and code understanding capabilities of the model.

\textbf{Rationale for Selection:}
\begin{itemize}
    \item Strong performance in code generation and reasoning tasks
    \item Suitable for educational and interview-style conversational systems
    \item Large context window supporting multi-turn dialogue
    \item Low-latency inference with Gemini 2.5 Flash
\end{itemize}

The project uses Gemini 2.5 Flash, which provides a balance between performance and computational efficiency. It is suitable for real-time chatbot applications requiring fast response generation.

Another important factor is cost efficiency. The availability of a free-tier quota through Google AI Studio significantly reduces operational cost, making it suitable for prototype development and initial deployment phases.

\subsection{n8n Workflow Automation Platform}

n8n is an open-source workflow automation platform that enables integration between APIs, services, and AI models through a node-based visual workflow system. It supports event-driven execution and allows developers to design data pipelines without tightly coupling logic into application code.\cite{n8n}

In the CodeRank system, n8n is used as the orchestration layer for AI-related workflows, particularly for chatbot processing and interaction management.

\textbf{Rationale for Selection:}
\begin{itemize}
    \item Supports rapid integration with APIs and AI services
    \item Provides visual workflow design for easier development
    \item Reduces coupling between frontend, backend, and AI logic
    \item Enables extensibility for future AI enhancements such as RAG or multi-agent systems
\end{itemize}

n8n serves as the middleware layer that connects the frontend system with the Gemini model and backend services in a structured and maintainable manner.
